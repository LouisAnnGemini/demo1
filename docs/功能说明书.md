# 员工管理系统 - 功能说明书

## 1. 系统概述
本系统是一个面向企业人事和管理层的员工综合管理与审核平台。系统旨在提供高效的组织架构管理、员工全生命周期信息维护、人事流程审批以及黑名单风险管控等功能，帮助企业实现人事管理的数字化与规范化。

## 2. 核心功能模块

### 2.1 组织架构管理
* **层级展示**：以树状图形式清晰展示“集团 -> 实体 -> 楼层 -> 区域 -> 品牌”等多级组织架构。
* **编制管理**：直观查看各品牌/区域的编制总数与已用编制数，支持通过进度条直观了解编制饱和度。
* **人员分布**：点击架构树节点，可快速过滤并查看该节点下的所有员工。

### 2.2 员工管理 (员工列表)
* **多维度筛选**：支持按姓名、工号、人员类型（正式、临时、厂聘）、所属实体、所属品牌、状态（未注册、未审核、已审核、已入职、已离职）以及入职时间段进行精准检索。
* **员工详情 (沉浸式弹窗)**：
  * **基础信息**：展示员工姓名、工号、手机号、角色标签、健康证状态等。
  * **附件信息 (智能分组)**：将员工附件智能归类为：身份证复印件、健康证、入职登记表、劳动合同、离职证明、银行卡复印件及其他附件。对缺失的核心资料提供红色的“未补充”高亮提示及快捷“上传”入口。
  * **任职履历**：以时间轴形式展示员工的入职、调动、离职等历史操作记录，方便追溯。
  * **离职与风险信息**：展示离职时间、离职证明、离职原因及黑名单状态。
* **编制操作 (绑编/解编)**：
  * 针对“已入职”状态的员工，提供快捷的绑编与解编功能。
  * **智能防呆逻辑**：若员工当前已有所属实体和品牌，则“绑编”按钮禁用，仅允许“解编”；若无所属实体和品牌，则“解编”按钮禁用，仅允许“绑编”。

### 2.3 审核中心
* **待办审核**：
  * 集中处理员工发起的各类业务申请（如：注册、入职、调动、离职等）。
  * 支持单条或批量“通过/驳回”操作。
  * **快捷查看**：在审核列表中点击“查看详情”按钮，可直接弹出该员工的完整资料详情界面，辅助HR进行审批决策。
* **审核日志**：
  * 记录所有历史审批流水。
  * 支持按目标人员、业务类型、时间段、申请人及审核状态进行多维追溯。

### 2.4 调动记录
* 完整记录员工在不同实体、品牌之间的调转轨迹。
* 提供调动类型、调动路径（从A调至B）、处理状态及操作时间的详细列表展示。

### 2.5 黑名单管理
* 集中展示因违规或其他原因被列入黑名单的人员名单，并展示详细的拉黑原因。
* **解除拉黑**：支持对黑名单人员进行“解除”操作。为保证操作严谨性，解除黑名单时必须强制填写解除原因，以留存审计记录。

## 3. 交互与体验设计
* **响应式与现代化UI**：采用 Tailwind CSS 构建，界面清爽，适配不同尺寸屏幕。
* **无缝弹窗体验**：员工详情、绑编/解编操作、解除黑名单等均采用沉浸式弹窗，减少页面跳转，提升操作连贯性。
* **状态可视化**：通过不同颜色和样式的标签（如绿色代表已入职/已通过，红色代表黑名单/驳回，橙色代表待审核）快速传递信息状态，降低认知负荷。
